Évolution d'une infrastructure d'entreprise

Pendant cette SAE, nous avions pour mission de répondre à une exigence visant à améliorer les modes de communication au sein d'une entreprise, tant entre les collaborateurs qu'entre ces derniers et les clients de l'entreprise.

Afin de relever ce défi, nous devions mettre en place un outil répondant aux spécifications suivantes :

- Le mode de communication envisagé devait prendre en charge aussi bien les échanges synchrones que les échanges asynchrones.

- Les conversations pouvaient se dérouler entre deux personnes (communication 1 vers 1), en petits groupes (discussions entre développeurs) ou à un niveau plus étendu (forum de discussion).

- Les échanges devaient être cryptés de bout en bout pour garantir la confidentialité.

- Le service de discussion devait être accessible via une application web, une application lourde ou une application mobile.
