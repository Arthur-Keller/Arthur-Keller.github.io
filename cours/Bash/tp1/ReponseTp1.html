Exercice 1.3 : Codes de retour et opérateurs logiques

Q16: Donnez une commande qui affiche le contenu de ~/R4.01/TP01 suivi du texte “C’est bien parti” si ce répertoire existe, et qui affiche seulement le texte “C’est mal parti” s’il n’existe pas.

Q17: Dans la page man de expr, trouvez comment obtenir la longueur d’une chaîne de caractères donnée. Ensuite, donnez une commande qui affiche le contenu de la variable d’environnement PWD si elle contient moins de caractères que la variable OLDPWD (et vice-versa).

# Définition des variables
EXPR1=$(expr length $PWD)
EXPR2=$(expr length $OLDPWD)

# Afficher contenue
expr $EXPR1 \< $EXPR2 && echo $PWD || echo $OLDPWD

Exercice 2.2 : Scripting

Q23: Créez un script test-quota qui réussit et renvoie le message Tout est ok. si l’utilisateur courant utilise moins de 5G dans son espace de stockage mais qui échoue et renvoie le message Pas assez d’espace ! dans le cas contraire.

#!/bin/bash

if [ $(quota -w | tail -n 1 | head -n 1 | cut -d " " -f 2) \< 5291456 ] 
then
    echo "Tout est ok"
else
    echo "Pas assez d'espace !"
fi
