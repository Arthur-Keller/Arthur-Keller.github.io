Weighted Shortest Job First · CoD = BV + TC + RR/OE · Job Size = Charge réelle (j/h)

BV — Valeur Métier (Priorité 1-5) TC — Criticité Temporelle (1-5) RR/OE — Réduction Risque (Rentabilité 1-5) Job Size = j/h réels

CoD  =  BV + TC + RR/OE
